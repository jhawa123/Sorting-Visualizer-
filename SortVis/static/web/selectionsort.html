Selection Sort

The selection sort algorithm sorts an array by repeatedly finding the minimum element from unsorted part and putting it at the beginning.

Python Code and complexity of Selection Sort


def stableSelectionSort(a, n): 	
	for i in range(n): 
		min_idx = i 
		for j in range(i + 1, n): 
			if a[min_idx] > a[j]: 
				min_idx = j 
		key = a[min_idx] 
		while min_idx > i: 
			a[min_idx] = a[min_idx - 1] 
			min_idx -= 1
		a[i] = key 

a = [4, 5, 3, 2, 4, 1] 
n = len(a) 
stableSelectionSort(a, n) 

for i in range(n):
	print(a[i],end=" ")

Complexity:

  1. Worst and Average Case Time Complexity: O(n*n)

  2. Best Case Time Complexity: O(n*n)

  3. Auxiliary Space: O(1)

  4. Sorting In Place: Yes

  5. Stable: No
