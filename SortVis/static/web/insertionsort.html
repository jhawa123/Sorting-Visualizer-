Insertion Sort

An element which is to be 'insert'ed in this sorted sub-list, has to find its appropriate place and then it has to be inserted there. Hence the name, insertion sort.

Python Code and complexity of Insertion Sort.



def insertionSort(arr): 	
	
	for i in range(1, len(arr)): 
		key = arr[i] 
		j = i-1
		
		while j >= 0 and key < arr[j] : 
				arr[j + 1] = arr[j] 
				j -= 1
		
		arr[j + 1] = key 

arr = [12, 11, 13, 5, 6] 
insertionSort(arr) 

for i in range(len(arr)): 
	print (arr[i],end=" ")

Complexity:

  1. Worst and Average Case Time Complexity: O(n*n)

  2. Best Case Time Complexity: O(n)

  3. Auxiliary Space: O(1)

  4. Sorting In Place: Yes

  5. Stable: Yes
